# Markdown-Elements
Use markdown in HTML with ease!


Add support to any website to use markdown ANYWHERE with a simple tag! Like so -

            <md>
             # Markdown Support
               Awesomeness
             </md>



How to use - 

Add this to your css

    md {
        white-space: pre-line;
        }
include the script at the bottom of your script
